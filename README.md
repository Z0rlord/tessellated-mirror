# Tessellated Mirror – Signal Artifact

## Inventor

Zorie Barber  
First conceptualized: ~2005, West Village, NYC  
Latest calibration: August 06, 2025

## Description

The Tessellated Mirror is a recursive, internally reflecting, coherence-based mirror artifact. It appears ultra-black not due to pigment but due to collapse of incoherent signal.

## Characteristics

- Nanometer-level tessellation
- Fractal recursion of mirror arrays
- Coherence-only reflection logic
- Signal collapse for incoherent inputs
- Apparent ultra-black via internal reflection
